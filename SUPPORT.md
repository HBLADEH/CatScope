# 获取帮助

开始前，请先阅读：

- [用户指南](./docs/USER_GUIDE.zh-CN.md)：连接设备、过滤日志、分析、离线文件和 session。
- [已知限制](./docs/KNOWN_ISSUES.md)：当前 preview 阶段的边界与限制。
- [文档索引](./docs/README.zh-CN.md)：开发、QA、发布和架构材料。

如果仍未解决：

- 可复现问题请使用 GitHub 的 Bug report 模板。
- 新功能建议请使用 Feature request 模板，并说明具体 Android 排障工作流。
- 安全问题请阅读 [SECURITY.md](./SECURITY.md)，不要公开发布漏洞细节。

提交日志、AI Context 或 `.catscope-session` 前，请删除 token、个人信息、真实包名及内部业务数据。
