# Known Issues

CatScope v0.6.0-preview is a lightweight Android Logcat debugging workbench, not a complete IDE. The following limitations are expected for this preview stage:

- CatScope is not an Android Studio replacement.
- Gradle Project Sync is not supported.
- Code editing is not supported.
- Module and variant selection for Build / Install / Launch is still basic.
- Launcher activity automatic parsing is not complete.
- External AI APIs are not integrated.
- AI Context is generated locally as Markdown only.
- Large `.catscope-session` files are not compressed or split.
- Vite may print a chunk size warning during frontend builds; this is currently accepted and does not block normal use.

See the main README files and user guides for current scope and setup details.
